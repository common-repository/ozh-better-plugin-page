== Description ==

**Better Plugin Page** brings a few valueable enhancements to the Plugin Management page: tiny icons, less clutter, quick action links. Handy for those who have 70+ plugins on their blog.

Check the [demo here](http://planetozh.com/download/plugins/better-plugin-page-demo/) and visit the official page of [Better Plugin Page](http://planetozh.com/blog/my-projects/wordpress-better-plugin-page/)

**Tip:** To keep track of this plugin's evolution and features, you are advised to subscribe to the [author's feed](http://planetozh.com/exit/feed)

== Installation ==

1. Unzip & upload the plugin directory inside your `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Play with it.
